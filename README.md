# M5Reflowとは

本キットは SANYOもといPanasonicのオーブントースターNT-T500ないしNT-T300に
組み込み、M5Stackで制御してリフローオーブンに改造する為のキットです。
本キット以外に別途改造元になる上記トースターと制御用のM5Stackが
必要となります。

組み込みについての解説は以下を参照して下さい。
http://www.ooishoo.org/wordpress/project/m5reflow/
現在小ロットの先行販売のみですが、今後委託販売を予定しております。

# リポジトリの構成

/src       M5Stack制御用プログラムソース
